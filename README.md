# AI 多人出行汇合点推荐助手

一个面向征文/作品展示场景设计的静态 Demo：

- Demo 页面：多人出行汇合点推荐
- 文章页面：参赛征文展示稿
- 部署方式：GitHub Pages

## 在线访问

- Demo: https://wozhi-xiaomai.github.io/tencent-map-ai-demo/
- Article: https://wozhi-xiaomai.github.io/tencent-map-ai-demo/article.html

## 项目亮点

- 围绕“AI + 地图”的真实多人决策场景
- 不是单人导航，而是多人汇合点公平推荐
- 强调可解释性、可视化和作品展示感
- 适合作为 CSDN 征文配套演示页

## 页面结构

- `index.html`：交互式 Demo 页面
- `article.html`：征文风格文章页
- `README.md`：仓库说明

## 可继续扩展

- 接入真实 POI 检索
- 接入真实路线规划
- 接入大模型解释排序
- 加入登录、收藏、分享、截图导出等功能
